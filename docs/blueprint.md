# **App Name**: CodeStudio

## Core Features:

- User Authentication: Implement user authentication using NextAuth.js with Google and GitHub providers.
- Profile Creation and Management: Allow users to create and manage their profiles, including updating information like username, bio, and profile picture.
- Code Snippet Sharing: Enable users to share code snippets with descriptions, tags, and the ability to specify the programming language.
- Code Snippet Feed: Display a feed of code snippets from all users, sorted by recency or popularity.
- Commenting and Discussion: Allow users to comment on code snippets and participate in discussions.
- Like and Save Snippets: Allow users to like code snippets, and save those snippets to a personal collection for later use
- AI Snippet Analysis: Use generative AI as a tool for analyzing shared snippets and adding tags if the user has not done so themselves

## Style Guidelines:

- Primary color: Midnight Blue (#2C3E50) for a professional and focused atmosphere.
- Background color: Dark Gray (#34495E) providing a clean backdrop that emphasizes content.
- Accent color: Electric Blue (#7DF9FF) for interactive elements and highlights, giving a vibrant contrast.
- Headline font: 'Space Grotesk' sans-serif for headlines and short amounts of body text; longer text should use Inter for body.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use modern, minimalist icons from a set like 'Lucide' to ensure clarity and consistency across the platform.
- Implement a responsive layout using Tailwind CSS grid and flexbox utilities to ensure the platform is accessible on all devices.
- Subtle transition effects for UI elements to enhance user experience without being distracting.
